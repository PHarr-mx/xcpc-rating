## 需求摘要

校内 XCPC 系列比赛的 Rating 统计与展示系统：

- 统计对象：**选手**（含多 OJ 账号）、**队伍**（1～3 人，以队员集合识别同一队）
- 数据来源三类：**正式比赛**、**校内训练赛**（`team_xcpc` / `solo_xcpc` / `oi` × Div1–3）、**OJ**
- 榜单：**仅正式赛** / **全部数据** 两种；时间维度：**生涯**、**赛年**、**赛季**
- Rating **计算公式待定**，设计预留插件接口
